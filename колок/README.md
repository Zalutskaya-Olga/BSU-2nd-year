# Проект Kolok1: Калькулятор факториалов

Этот проект реализует калькулятор для вычисления факториалов на языке C++ с использованием CMake в качестве системы сборки.

## Структура файлов

- `CMakeLists.txt` - конфигурация сборки CMake
- `kolok1.h` - заголовочный файл с объявлениями классов
- `kolok1.cpp` - реализация основных функций
- `kolok1_gtest.cpp` - модульные тесты с использованием Google Test

## Сборка и запуск

### Требования
- CMake 3.10 или выше
- Компилятор C++ с поддержкой C++98
- Google Test

### Сборка
```bash
mkdir build
cd build
cmake ..
make
```

### Запуск
```bash
# Основная программа
./kolok1

# Тесты
./kolok1_test
```

## Тестирование

Проект включает комплексные модульные тесты, проверяющие:
- Корректность вычислений факториалов
- Обработку граничных случаев (n=0, n=1)
- Обработку ошибок (отрицательные значения, переполнение)
- Работу кэширования в OptimizedFactorialCalculator

## Время разработки

**Затраченное время на разработку: около 1-1.5 часа**

Это время включает:
- Проектирование архитектуры классов
- Реализацию основной логики
- Написание модульных тестов
- Отладку и проверку корректности работы
- Создание демонстрационной программы

Проект демонстрирует эффективное использование принципов объектно-ориентированного программирования и практик тестирования для создания надежного и поддерживаемого кода.
